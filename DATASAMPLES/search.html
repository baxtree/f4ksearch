Downloadable Datasets

This page provides a subset of the full Fish4Knowledge video and fish analysis dataset, which is about 200 Tb in size. See ***URL*** for downloading options from the full video data.

There are two interesting subsets of the data that are provided here:

  • 1) All Years: a 10 minute video clip from all cameras taken at 12:00 every day in the project Oct 1, 2010 - July 31, 2013, giving approximately 10K video clips. This data allows analysis of fish patterns over annual cycles and comparison between sites.
  • 2) Full Day: all 720 video clips from the 10 cameras taken from 06:00 - 18:00. This data allows analysis of fish patterns over a full day period and comparison between sites.

There are 4 sites with different numbers of cameras:

  • 1 - NPP: 4 cameras
  • 2 - HoBiHu harbour: 2 cameras
  • 3 - Lanyu/Orchid Island - 2 cameras
  • 4 - Kenting Harbour - 2 cameras
  • All Years
  • Full Day

The "All Years" dataset consists of video from all cameras at the 4 sites, for the time period 12:00-12:10, for all video capture days a single date: April ***, 2011. (Except for some videos, where there were technical problems.) This data allows analysis of fish patterns over annual cycles and comparison between sites. There are approximately 10K video files each about 50 Mb long (Oct 1, 2010 - July 31, 2013 - 1035 days * 10 cameras).

Select the start and stop days desired by clicking in the 'Day' boxes to pop-up a calendar. Select the camera(s) and sites. Select multiples sites/cameras by using the CTRL-click on the desired selections. Not all sites had 4 cameras (see above for more information).

When you click 'Download', the system will produce 2 or more 'zip' files, which will appear as clickable links below, along with the file sizes. One zip file will contain all selected SQL files. The other zip files will contain up to 5GB of video files in each download.

The "Full Day" dataset consists of video from all cameras at the 4 sites, for all time periods (6:00 - 18:00, the daylight hours), but for only a single date: April ***, 2011. (Except for these videos, where there were technical problems: ***.) This data allows analysis of fish patterns over a full day period and comparison between sites. There are 720 videos (10 cameras * 12 hours * 6 10 minute clips).

Select the start and stop times desired, as well as which camera(s) and sites. Select multiples sites/cameras by using the CTRL-click on the desired selections. Not all sites had 4 cameras (see above for more information).

When you click 'Download', the system will produce 2 'zip' files, which will appear as 2 clickable links below, along with the file sizes. The zip files will contain all selected SQL and video files in one download.
